# 📜 Квиз по истории России

Интерактивный тренажёр для подготовки к экзамену. **234 вопроса** в 22 темах:
Древняя Русь → Российская империя → СССР. Неверный ответ — тема раскрывается.

Работает офлайн, без сборки и зависимостей. Можно установить на телефон как приложение (PWA).

## Файлы
| Файл | Назначение |
|------|-----------|
| `index.html` | приложение (открывается по корню сайта `/`) |
| `questions.js` | база вопросов — её и редактируй, чтобы добавлять вопросы |
| `manifest.webmanifest` | иконка/название при установке на телефон |
| `.htaccess` | настройки Apache-хостинга (кодировка, MIME, кэш) |

## Как выложить на reg.ru

`index.html` автоматически открывается по корню сайта — посетитель видит просто `твой-домен.ру`, без `/index.html`.

**ВАЖНО:** загружай файлы вместе со скрытым `.htaccess`.

### Способ 1. Через панель (Менеджер файлов) — проще всего
1. Зайди в **Личный кабинет reg.ru → Мои услуги → твой хостинг → Перейти в панель управления** (ISPmanager).
2. Открой **«Менеджер файлов»** и зайди в папку сайта — это корневая папка домена
   (обычно `/www/твойдомен.ру/` или `.../data/www/твойдомен.ру/`).
3. **Удали** дефолтные файлы-заглушки, если они там есть (`index.html`, `index.php`, `parking`).
4. Загрузи сюда: `index.html`, `questions.js`, `manifest.webmanifest`, `.htaccess`.
   *(Можно загрузить ZIP всей папки и распаковать кнопкой «Извлечь».)*
5. Открой `https://твойдомен.ру` — работает.

### Способ 2. Через FTP (FileZilla)
1. В панели reg.ru возьми **FTP-данные**: сервер, логин, пароль (раздел FTP-пользователи).
2. В FileZilla подключись и зайди в корневую папку домена (см. выше).
3. Перетащи туда все файлы, включая `.htaccess`
   (в FileZilla включи показ скрытых файлов: *Сервер → Принудительно отображать скрытые файлы*).

> Если файлов `.htaccess` не видно в Finder на Mac — нажми **Cmd+Shift+.** (точка), чтобы показать скрытые файлы.

## Свой домен и HTTPS
- Домен на reg.ru обычно уже привязан к хостингу. Если нет — в панели: **Сайты → Привязать домен**.
- Бесплатный SSL (HTTPS): в панели хостинга включи **Let's Encrypt** для домена.

## Редактирование вопросов
Открой `questions.js` — формат очевиден:
```js
{
  q: "Текст вопроса?",
  options: ["вариант 1", "вариант 2", "вариант 3", "вариант 4"],
  correct: 0,            // индекс правильного ответа (0 = первый)
  explain: "Объяснение, которое раскроется при ответе."
}
```
Варианты ответов и сами вопросы перемешиваются автоматически.
